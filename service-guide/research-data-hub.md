# Open Source Research Data Hub

To further lower the barrier to data access and accelerate open source research implementation, OSS-Compass has launched the "Open Source Research Data Hub Service", aiming to build a highly available, trustworthy, and extensible data acquisition platform. This service is not just a data provision tool, but a key step in our effort to drive open source research toward systematization, transparency, and automation.

No matter what role you play in the development process, you can benefit from using this service:

- **Researchers:** Obtain standardized historical data to reproduce experiments or build models
- **Developers/Platform Providers:** Real-time access to open source project dynamics through API
- **Policy Researchers/Community Operators:** Gain insights into open source ecosystem structure and development trends
- **Enterprise R&D Managers:** Analyze their participation and influence in the open source field

## Features

The "Open Source Research Data Hub Service" launched by OSS-Compass has the following advantages:

1. **Unified Access Method:** Provides standardized data access through REST API and archive data download interfaces.

2. **Rich Data Resources:** Covers core dimensions of open source projects including commit, issue, PR, release, star, organization activity, supporting indicator modeling and research analysis: Can serve as the data foundation for CHAOSS indicator evaluation, influence modeling, and other work.

3. **Promote Academic and Industry Collaboration:** Help universities, research institutions, and open source companies share basic data, promoting open source governance and development research.

## Operation Guide

**Step1:** First, visit the OSS-Compass website: [https://oss-compass.org](https://oss-compass.org), click "All Services" in the navigation bar, select "Open Source Research Data Hub Service" product homepage in the popup, or mirror sites: Institute of Software, Chinese Academy of Sciences, or Open Source Analysis Lab, School of Computer Science, Peking University (click any one), to enter the service homepage.

![](media/image21.png)

**Step2:** Get Data through REST API

Click "REST API" on the service homepage to enter the detail page, where you can view interface authentication methods, all supported interfaces, parameter descriptions, example requests, and response formats.

![](media/image22.png)

![](media/image23.png)

**Step3:** Download Data Packages through Archive Data

Click "Archive Data Download" on the service homepage to enter the download page, where you can choose to download quarterly or annual open source ecosystem trend insight archive data.
