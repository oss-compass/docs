# Service Guide

This guide introduces the core services provided by OSS Compass platform, helping users understand and utilize various evaluation and analysis tools.

- **Open Source Trend Insights Service**: Provides comprehensive insights into open source ecosystem trends and community health status.
- **Open Source Software Selection Evaluation Service**: Helps enterprises and developers evaluate and select suitable open source software.
